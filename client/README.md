# Reddit Clone - Front End

## Installation

- `git clone` the project
- `cd` into `/client`
- Create a `.env` file like below
- `npm i` dependencies
- `npm run dev`
- Open your browser to [http://localhost:4000/](http://localhost:4000/)

## `.env` file

```
API_URL=[urlHere]
```
